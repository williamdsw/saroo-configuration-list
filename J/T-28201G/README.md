# Kaitou Saint Tail (J)

SAROO's attributes configuration for this game.

### Usage

- Add these 3 lines on **Solution** section or from the [text file (.txt)](./config.txt) to your saroocfg.txt

### Problems

- It does shows a black screen after booting from menu.
