# Batman Forever: The Arcade Game (Japan)

SAROO's attributes configuration for this game.

### Usage

- Add these 3 lines on **Solution** section or from the [text file (.txt)](./config.txt) to your saroocfg.txt

### Problems

- Graphical glitches in the opening scene.

### Solution

[Based on this file on SAROO's official GitHub page](https://github.com/tpunix/SAROO/blob/6d213af4a484f2f6e5d310d0eb8cc070f5a45775/doc/saroocfg.txt#L115)

<pre># Batman Forever: The Arcade Game (Japan)
[T-8118G   V1.000]
play_delay = 100000
sector_delay = 4200</pre>
